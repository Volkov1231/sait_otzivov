Установить модуль джанго
pip install django

создать джанго-проект. Внутри активной папки будет создана папка со стандартными файлами. Рекомендуется назвать проект "ядро" и потом корневую папку переименовать как надо
django-admin startproject имя_проекта

Первый запуск сервера 
В папке, где лежит manage.py выполнить команду
python manage.py runserver

Если запустилось , создаём приложение. Это - рабочая часть сайта, которая устанавливается поверх стандартных файлов, и где пишется основной команду
python manage.py startapp название_проекта

Подключить приложение к ядру
В папке ядро есть файл settings.py
Там ищем INSTALLED_APPS
Добавляем строчку:
приложение.apps.ПриложениеConfig
(приложение - название папки)
(apps - название файла внутри)
(ПриложениеConfig - название класса внутри него)

для проверки следует запустить сервер



Чтобы поменять часовой пояс на московский в settings.py надо поменять строку
TIME_ZONE = 'Europe/Moscow'
