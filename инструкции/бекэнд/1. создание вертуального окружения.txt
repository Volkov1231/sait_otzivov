Вертуальное окружение - это изолированная среда, не зависящая от системы. Запуск в нём равносилен запуску проекта на новом компьютере.

СОЗДАНИЕ
Создаётся командами в терминале, внутнри активной папки (которая пишетс\я желтым цветом, переходить по папкам нужно командой cd)

При создании обращаемся к нужной версии питона (на которой хотим запустить проект). Например, python3.9
Если написать просто python, будет выбрана стандартная версия

дальше нужно написать
-m venv venv
Первое venv - это команда для создания виртуального окружения (вирт. окружения)
Второе venv - имя папки. ОНо может быть любым, но рекомендуется использовать venv


в итоге полдучается такая строчка
python -m venv venv
